# Blockchain-Based Secure Medical Records Sharing Platform

## Project Overview
This project is a secure medical record sharing system that uses blockchain technology to protect patient data and provide secure access to authorized users. It ensures data integrity, privacy, and transparency while preventing unauthorized modifications.

## Features
- Secure storage of medical records
- Blockchain-based data integrity
- SHA-256 hashing for security
- User authentication and authorization
- Secure record sharing between patients and healthcare providers
- Tamper-resistant medical data

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite / MySQL
- Blockchain
- SHA-256

## Project Workflow
1. User registers and logs in.
2. Medical records are uploaded securely.
3. SHA-256 hash is generated for each record.
4. Blockchain stores the hash to ensure data integrity.
5. Authorized users can securely access and verify records.

## Benefits
- Improved security
- Data integrity verification
- Privacy protection
- Decentralized record management
- Reduced risk of data tampering
